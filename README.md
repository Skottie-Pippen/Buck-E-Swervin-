# Buck-E-Swervin-

Game Dev

## Overview

**Buck-E-Swervin-** is a game development project written primarily in C. The project demonstrates core game development techniques, including game loop management, input handling, and graphics rendering.

## Features

- Written mostly in C for performance and portability
- Includes sample Makefile for building the project
- Designed for extensibility and learning purposes
- Modular structure for easy understanding and updates

## Getting Started

### Requirements

- GCC or any compatible C compiler
- Make utility

### Building the Game

Clone the repository and build using `make`:

```bash
git clone https://github.com/Skottie-Pippen/Buck-E-Swervin-.git
cd Buck-E-Swervin-
make
```

This will compile the source code and generate the executable.

### Running the Game

After building, run the executable (name may vary according to your Makefile):

```bash
./buck-e-swervin
```

## Repository Structure

- `src/` — Main source files 
- `include/` — Header files
- `Makefile` — Build instructions
